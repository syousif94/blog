For maps on space constrained devices, clustering nearby markers together makes it easier for users to interact with the map and helps improve performance by reducing the number of views on screen. I'm going to go over how to use [ClusterKit](https://github.com/hulab/ClusterKit) with custom view subclasses to produce elegant maps for iOS.
